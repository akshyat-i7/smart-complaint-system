# Smart Complaint Management System

## 📌 Overview
This project is a web-based complaint management system built using Flask and SQLite. It allows users to submit complaints and track their status. The system also predicts complaint priority (High, Medium, Low) using a simple machine learning approach.

## 🚀 Features
- User login system
- Add complaints
- View all complaints
- Automatic priority classification
- Clean UI with CSS

## 🛠️ Tech Stack
- Python (Flask)
- SQLite
- HTML/CSS

## ⚙️ Setup Instructions
1. Clone the repository
2. Install dependencies:
   pip install flask
3. Run the application:
   python app.py

## ▶️ Usage
- Open browser: http://127.0.0.1:5000/
- Login using:
  - Username: admin
  - Password: admin
- Add and view complaints
